# Responsive Header

A fully responsive website header built with **HTML, CSS, and JavaScript**.  
Includes a desktop navigation, mobile hamburger menu, and smooth slide-in mobile menu animation.

---

## Features

- **Responsive design**: Works on all screen sizes (desktop, tablet, mobile)  
- **Mobile hamburger menu**: Slide-in navigation for mobile devices  
- **Sticky header**: Header stays at the top while scrolling  
- **No-scroll lock**: Prevents page scrolling when mobile menu is open  
- **Smooth animations**: Hamburger menu transitions and mobile menu slide-in  
- **Clean and modern UI**
